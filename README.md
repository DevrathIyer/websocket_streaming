# Websocket Streaming for Pi in the Sky
Webcam live streaming with WebSockets  
  
[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy)

## Instalation
```text
npm install
```
## Usage
```text
npm run start
```

- Go to */client* and a live feed will appear
